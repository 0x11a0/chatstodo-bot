# ChatsToDo Bot

## Set up

1. Install virtualenv

    ```
    pip install virtualenv
    virtualenv venv
    ```

1. Start virtualenv

    For macOS
    ```
    source ./venv/bin/activate
    ```

    For Windows
    ```
    cd venv/bin
    activate.bat
    ```

1. Install requirements

    ```
    pip install requirements.txt
    ```
